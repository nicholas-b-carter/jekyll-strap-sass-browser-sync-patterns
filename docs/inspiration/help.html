Creating a report

By using a report, you can summarize, analyze, explore, and present a summary of the application data sources. A report is especially useful to help visualize data from different perspectives and compare samples of similar data.

Steps to create a report

  1. Define your data source.
  2. Choose a report format or create a custom report template.
  3. Select a theme.
  4. Generate and preview the report.
  5. Publish the report.
